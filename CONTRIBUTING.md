# Contributing to Squirrelsoft Marketplace

Thank you for your interest in contributing to the Squirrelsoft Dev Tools Marketplace! This marketplace is a community-driven collection of tools, skills, and agents for Claude Code.

## How to Contribute

### Adding a New Plugin

1. **Fork the repository** and create a new branch for your plugin
2. **Add your plugin** to the appropriate section in `README.md`
3. **Follow the format** of existing entries:
   ```markdown
   ### Plugin Name
   Brief description of what the plugin does.
   ```bash
   /plugin install plugin-name@owner
   ```

   **Includes:**
   - Feature 1
   - Feature 2

   [View on GitHub](https://github.com/owner/plugin-name)
   ```
4. **Test your plugin** thoroughly before submitting
5. **Submit a pull request** with a clear description of what your plugin does

### Plugin Requirements

- Must be a valid Claude Code plugin, skill, or agent
- Must have clear documentation in its own repository
- Must follow best practices for Claude Code development
- Must be properly licensed
- Should not duplicate existing functionality

### Reporting Issues

If you find a problem with a plugin listed in the marketplace:
1. Check if the issue exists in the plugin's own repository
2. If it's a marketplace-specific issue (like incorrect information), open an issue here
3. Provide clear details about the problem

### Suggesting Improvements

We welcome suggestions for improving the marketplace itself:
- Better organization or categorization
- Additional metadata for plugins
- Improved discovery features
- Documentation improvements

## Code of Conduct

Please note that this project follows a Code of Conduct. By participating, you agree to uphold this code.

## Questions?

Feel free to open an issue for questions or reach out via [squirrelsoft.dev](https://squirrelsoft.dev).
