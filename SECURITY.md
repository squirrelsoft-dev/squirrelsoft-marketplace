# Security Policy

## Supported Versions

This marketplace serves as a directory of Claude Code plugins. Security updates apply to the marketplace repository itself and its documentation.

| Version | Supported          |
| ------- | ------------------ |
| Latest  | :white_check_mark: |

## Reporting a Vulnerability

### For Marketplace Repository Issues

If you discover a security vulnerability in the marketplace repository itself (not in a listed plugin), please report it by:

1. **DO NOT** open a public issue
2. Email security concerns to the contact information at [squirrelsoft.dev](https://squirrelsoft.dev)
3. Include:
   - Description of the vulnerability
   - Steps to reproduce
   - Potential impact
   - Suggested fix (if any)

We will respond within 48 hours and work with you to understand and address the issue.

### For Plugin-Specific Issues

If you find a security issue in a plugin listed in this marketplace:

1. Report the issue directly to that plugin's repository
2. Follow the security policy of that specific plugin
3. Optionally, let us know so we can add a security notice to the marketplace listing

## Security Best Practices for Plugin Authors

When submitting plugins to this marketplace, we recommend:

- Include a SECURITY.md file in your plugin repository
- Keep dependencies up to date
- Follow secure coding practices for Claude Code plugins
- Clearly document any permissions or access requirements
- Respond promptly to security reports

## Disclosure Policy

- Security issues will be addressed promptly
- Fixes will be released as soon as they are validated
- Credit will be given to reporters who wish to be acknowledged
- We will coordinate disclosure timing with reporters when appropriate
